# 爱物理 - 力与运动仿真平台（NEXT 仿真力学模块复刻）

复刻自 [爱物理](http://www.iwuli.cn/forcemotion) 力与运动仿真平台的 **NEXT 仿真 / 力学模块**，使用 Next.js + TypeScript + Canvas 2D 重新实现其物理仿真编辑器与实时仿真引擎。

## 技术栈

| 层 | 选型 |
|---|---|
| 框架 | Next.js 14 (App Router) |
| 语言 | TypeScript 5 |
| 样式 | Tailwind CSS 3 |
| 状态 | Zustand |
| 渲染 | Canvas 2D（自研渲染引擎） |
| 物理 | 自研力学求解器（半隐式欧拉 + 子步长） |

> 原站使用 React SPA + WASM(Ceres 优化库) + 文档/视图架构。本复刻用纯 JS 物理引擎实现等效的力学仿真，无需 WASM。

## 功能特性

### 编辑器（对应原站 forcemotion 界面）
- **左侧绘图工具栏**：选择、质点(粒子M)、弹簧(T)、地面/管模型、辅助点(P)、辅助线、文本框
- **参数面板**：全局重力(开关+g值)、考虑碰撞(开关)、空气阻力、滑动条配置、自定义参数(g/pi)、视图配置、场景统计
- **顶栏**：模块选择器(NEXT/动力学/电路/光学/万有引力/板块)、仿真名称(可编辑)、预设实验、清空、教程、登录
- **右侧属性面板**：选中对象后编辑外观属性与物理属性（质量/速度/劲度系数/摩擦因数/颜色等）
- **底部播放控制**：进入仿真环境、播放/暂停、返回编辑、时间显示、速度调节(0.25×~4×)、函数图像开关

### 物理仿真引擎
- 半隐式欧拉积分 + 4 子步长（稳定性）
- 全局重力、空气阻力
- 弹簧力（胡克定律 + 阻尼），弹簧可连接质点或固定点
- 质点-地面碰撞（位置修正 + 恢复系数 + 摩擦衰减）
- 质点-质点碰撞（冲量法，动量守恒）
- 实时轨迹绘制

### 渲染引擎
- 世界坐标系 ↔ 屏幕坐标转换（y 轴向上）
- 自适应背景网格、带刻度数字的坐标轴
- 质点（径向渐变 + 速度箭头 + m 标识 + 轨迹）
- 弹簧（螺旋线）、地面（斜线纹理 + μ 标注）、辅助点/线/文本
- 滚轮缩放、中键/右键平移

### 函数图像
- 选择观测质点，输入 Y 轴表达式（x/y/v/vx/vy/能量等）
- 实时绘制物理量随时间变化曲线，支持预设快捷表达式

### 预设力学实验（8 个）
| 实验 | 验证要点 |
|---|---|
| 自由落体运动 | 匀加速直线运动 |
| 平抛运动 | 水平匀速 + 竖直匀变速 |
| 斜抛运动 | 抛物线轨迹（最高点 4.42m vs 理论 4.45m） |
| 竖直弹簧振动 | 简谐运动（周期振荡） |
| 水平弹簧振动 | 简谐运动 |
| 弹簧双阵子 | 耦合振动 + 动量守恒（3.000） |
| 两球正碰 | 动量守恒（5.0 = 5.0） |
| 抛体运动对比 | 不同初速度抛物线对比 |

## 交互方式

| 操作 | 效果 |
|---|---|
| 选择工具 + 点击 | 选中对象，右侧显示属性 |
| 选择工具 + 拖动 | 移动对象 |
| 质点工具 + 拖动 | 放置质点，拖动方向设初速度 |
| 弹簧工具 + 拖动 | 绘制弹簧，端点靠近质点自动连接 |
| 地面/辅助线 + 拖动 | 绘制线段 |
| 滚轮 | 缩放视图 |
| 中键/右键拖动 | 平移视图 |

## 项目结构

```
src/
├── app/
│   ├── layout.tsx          # 根布局
│   ├── page.tsx            # 主页面（组合所有组件）
│   └── globals.css         # 全局样式
├── components/
│   ├── TopBar.tsx          # 顶栏（模块选择/实验名/操作）
│   ├── Toolbar.tsx         # 左侧绘图工具栏
│   ├── ParamPanel.tsx      # 左侧参数面板
│   ├── SimCanvas.tsx       # 画布（渲染循环+鼠标交互）
│   ├── PropertyPanel.tsx   # 右侧属性面板
│   ├── BottomBar.tsx       # 底部播放控制
│   ├── ChartPanel.tsx      # 函数图像
│   ├── ExperimentDrawer.tsx# 预设实验抽屉
│   └── Icons.tsx           # SVG 图标
└── lib/
    ├── types.ts            # 类型定义
    ├── store.ts            # Zustand 状态管理
    ├── physics.ts          # 物理仿真引擎
    ├── render.ts           # Canvas 渲染引擎
    └── experiments.ts      # 预设实验数据
```

## 开发与运行

```bash
pn install          # 安装依赖
pn dev              # 开发模式 (http://localhost:3000)
pn build            # 生产构建
pn start            # 生产运行
```
